Library Management System

A Django-based library management platform that empowers authors to manage their literary catalog with full CRUD functionality. The system features user authentication, book cataloging, and format management in a clean, intuitive interface.

Key Features
  • Author Authentication: Custom user system extending Django's AbstractUser with additional fields (pseudonym, birthday)
  • Book Management: Complete CRUD operations for books with pricing, format categorization, and multi-author support
  • Literary Format Organization: Manage and categorize different literary formats
  • Search & Filtering: Advanced book search functionality with title-based filtering
  • Session Tracking: Visit counter implementation using Django sessions
  • Responsive Pagination: Optimized book listing with configurable page size
